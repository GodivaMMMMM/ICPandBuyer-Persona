## 裸机实例地域的高可用、回国优化、三层网络接入是什么意思？
## What do High Availability, China-optimized, and Cloud Router Enabled mean for Bare Metal instance location?
- 城市与区域
单个城市下包含一至多个区域，不同区域支持的功能不同，可选机型不同。
如果您对功能无特定要求，可选择随机区域，方便查看所选城市下支持的所有机型。
- 高可用
高可用指网卡高可用，将服务器上两个或更多的物理网卡绑定成一个逻辑上的网卡，实现链路冗余，用于对网络稳定性要求较高的场景。
- 回国优化
回国优化指的是某些城市支持 CN2，即电信 CN2 GIA 回国优化线路，是目前最快最稳定的回国线路，能够快速从海外访问中国。目前支持 CN2 的城市有：洛杉矶、香港和东京。
- 三层网络接入
不同区域不同型号的服务器，接入骨干网的能力不同。有些支持二层网络接入，有些支持三层网络接入，即支持创建 VPC。如您有相关组网需求，请选择支持相应功能的区域的服务器。
即将推出更多城市支持以上功能，敬请期待

## 裸机实例的公网带宽计费模式有哪些类型？
## What are the public network billing models for Bare Metal instances?
- 流量计费
支持按量付费购买流量。每小时结算一次每 GB 的流量费用，当余额不足时，将停止服务和计费。
- 固定带宽计费
如果您的业务稳定，只有少数突发流量，则可以设置带宽上限，使用无限流量。
- 合并突发 95 计费
您可以先使用服务，然后再按月支付费用（如存在保底带宽，则该部分需预付费）。在日历月的每个有效日，系统会每 5 分钟对一个带宽组中的所有实例的峰值带宽进行采样。 一个月结束后，前 5% 的样本会被丢弃，剩余样本中的最高值作为您的实际带宽使用量。

暂时不加入任何子网，待实例创建完成后，再规划网络，配置内网网络结构。

## 弹性算力基础磁盘和标准磁盘有什么差异？
## What is the difference between Basic and Standard disks for Elastic Compute?

标准型 NVMe SSD 具备更高的速度和更强的可靠性，适用于更高负载的应用场景。其可靠性高达 99.99%，并配备三重（3x）备份机制，是对数据完整性和持续运行要求严格的高性能计算环境中的可靠选择。

## 弹性算力实例的公网计费方式是什么？
## What are the public network billing models for Elastic Compute instances?
- 流量计费

## ZEC弹性IP来源有哪些？
## What are the IP sources of Elastic Compute elastic IPs?
- 默认 IP 地址池
默认 IP 地址池是一个供公共访问的弹性 IP 资源池，涵盖所有地区。您可以选择一个地区来创建弹性 IP。释放的 IP 会返回池中，若未被占用，可以重新创建。默认 IP 地址池适用于对 IP 数量需求较少的场景。
- IP 地址池
IP 地址池是一个供私有访问的弹性 IP 资源池，包含多个 CIDR 地址块，随着 CIDR 地址块的创建自动生成。一个用户在一个区域内只能拥有一个 IP 地址池。您可以租用 IP 或自带 IP（BYOIP）并享有独占访问权限。适用于对大量连续 IP 的需求和管理场景。如有需要，请联系我们。

## 弹性算力实例的弹性IP有哪些网络类型？
## What elastic IPv4 network types are available for Elastic Compute?
非中国大陆境内区域：

- 优质BGP/PremiumBGP IP
优质BGP/PremiumBGP IP 是一种通过 BGP（Border Gateway Protocol）路由协议实现的弹性 IP 地址。它允许用户自主管理其网络路由，提供了高度的灵活性和可靠性，适用于需要可扩展性和高可用性的网络应用场景。

- 本地 IP
本地 IP 地址由本地运营商提供，且其注册地址与服务器机房所在国家一致。适用于对 IP 地址有严格限制的应用场景，例如跨境电商，使用本地 IP 地址可更顺利完成注册和操作。

- 回国优化 IP
回国优化 IP 是指基于CN2 GIA（中国电信下一代承载网），这是目前速度最快、稳定性最高的中国线路，可帮助您从海外高效访问中国。目前支持 CN2 的城市包括洛杉矶、香港和东京。

- 标准BGP/StandardBGP IP
标准BGP/StandardBGP IP指的是通过多个 Tier 1 骨干网络（如 Cogent Communications、GTT Communications）提供高速互联网与数据传输服务，面向通用连接场景，在主要全球区域实现成本优化的网络接入提供。
