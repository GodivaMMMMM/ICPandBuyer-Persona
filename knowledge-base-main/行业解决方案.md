# 游戏行业 — 实时战斗游戏

### ** 客户背景**

- 适用于实时对战、操作频繁、延迟敏感的游戏类型。
- 玩家分布全球，不同地区服务器部署与网络覆盖差异明显。

### ** 客户需求**

1. **服务全球用户**：节点需覆盖多区域，兼顾不同国家政策。
2. **保证超低时延**：玩家操作需无卡顿、技能释放无延迟、画面无撕裂。
3. **灵活拓展**：支持快速在各区域新增节点、灵活部署及网络调优。

### **解决方案**

- **边缘裸金属服务器/弹性计算部署战斗服**：25ms 内超低接入时延、支持多种机型、快速开通。
- **SDN 专线连接核心服与战斗服**：高安全性、高可靠性、延迟优化。
- **裸金属/弹性计算部署核心服**：性能强、资源稳定，支撑高并发。
- **ZDP 安全防护**：支持四层 / 七层 DDoS 防护、全球清洗节点。

### ** 实施架构**

核心节点（新加坡）——登录、版本、商城、聊天
│
├── 战斗节点（雅加达、曼谷、新加坡）
│        ↕ 玩家（印尼、泰国、新加坡）
└── CDN 回源：游戏下载/更新

# 游戏行业 — 休闲游戏

### **客户背景**

- 适用于 MMORPG、策略、卡牌类游戏；并发量高、计算压力大。

### **客户需求**

1. **全球/区域同服架构**：并发数高、单节点计算能力有限。
2. **提升玩家体验**：需优化登录速度、操作响应、掉线率。
3. **弹性与安全**：应对突发扩容、DDoS 攻击等情况。

### ** 解决方案**

- **部署 ZGA 全球加速服务**：通过 GSLB 智能调度、边缘加速节点、全球骨干互联，提供低延迟访问。
- **ZDP 防护产品**：提供全球 DDoS 防护能力，七层安全防御。
- **裸金属服务器/弹性计算部署游戏主服**：高性能、支持物理机与虚拟化混合部署。

### ** 实施架构**

全球玩家 → ZGA 全球加速节点 → ZGA Backbone → 游戏主服（登录/商城/排行/数据库）

# 游戏行业 — 游戏加速器

### **客户背景**

- 用于游戏加速器厂商建设海外专线节点。
- 服务对象为跨国对战、海外服访问的玩家群体。

### **客户需求**

1. **提升玩家体验**：降低登录延迟、减少跨国网络不稳定。
2. **灵活高效部署**：快速满足新游戏需求、活动流量峰值。

### **解决方案**

- **裸金属或弹性计算 部署边缘接入点**：高性能 CPU/内存配置，快速弹性。
- **IPT 连接边缘用户**：国内节点支持 BGP / 三线三 IP 架构，海外节点一跳接入。
- **SDN 高可用海外线路**：德、美、日韩多方向出海专线，双线路主备保障。

### ** 实施架构**

国内玩家 → 加速接入点（PEK/SHA/CAN）
       │ 专线互联
      ▼
边缘节点 → 外服游戏服务器（多出口）

# RTC行业 — 音视频解决方案

### **客户背景**

- 适用于 RTC 客户（音视频、会议、直播、互动类）。
- 目标为搭建全球低时延、高可用的骨干通信网络。

### ** 客户需求**

1. **低延迟要求**：语音、视频通话无卡顿。
2. **多云互备**：云间专线互通，边缘与核心节点稳定互备。
3. **多区域访问**：覆盖东南亚、欧美、中东非。

### ** 解决方案**

- **SDN 专线连接公有云与边缘节点**：确保超低时延和高 SLA。
- **边缘节点 IPT 能力**：270+ 全球节点资源，BGP 接入就近路由。
- **弹性扩展**：可按需部署裸金属、快速交付专线。

### **实施架构**

主播 A / B 推流 → 实时通信网络 → 核心节点（认证/监控/调度）
↓
边缘节点（东南亚 / 中东非 / 欧美）
↓
观众端拉流播放（端到端延迟 < 500ms）

# CDN与公有云 — CDN行业部署方案

### ** 客户背景**

适用于互联网及媒体公司在海外建设 CDN 节点，以提升用户访问体验和内容分发效率。

### ** 客户需求**

1. **服务边缘用户**：覆盖本地群体、应对媒体内容多样化。
2. **提升运营商覆盖质量**：降低带宽成本、优化跨运营商网络。
3. **节点灵活扩容**：满足音视频、大文件分发等高并发需求。

### **解决方案**

- **部署边缘数据中心**：靠近用户、支持 BGP 出口及多 ISP 节点；
- **采用裸金属服务器**：高性能、灵活扩容、快速交付；
- **ZDP 防护产品保障 CDN 节点安全**：提供四层/七层防护与 Anycast 架构。

### **实施架构**

用户 → DNS解析 → 边缘节点 Cache集群 → 源站
（支持多层缓存、本地 BGP 节点、Anycast 网络）

# CDN与公有云 — 云厂商边缘国家覆盖方案

![image.png](%E8%A1%8C%E4%B8%9A%E8%A7%A3%E5%86%B3%E6%96%B9%E6%A1%88%2028be10c7659a80df8967da64b6c0037f/image%205.png)

### ** 客户背景**

适用于云服务商或大型互联网厂商在边缘国家（如拉美、东南亚、非洲等）优化云接入与区域覆盖。

### ** 客户需求**

1. 单核节点覆盖能力有限，需提升区域访问质量；
2. 跨国骨干高峰期拥塞，运营商间互联不稳定；
3. 本地建设周期长、成本高、运维复杂。

### ** 解决方案**

- **结合 SDN + IPT 产品**，提供一站式边缘国家覆盖优化；
- **SDN 专线** 提供低时延、稳定骨干连接；
- **IPT 直连** 当地主流运营商 Eyeball 网络，实现一跳接入；
- **支持 Anycast**，部署快速、终端无感知。

### ** 实施架构**

公有云主站 ↔ Zenlayer SDN Backbone ↔ 边缘国家 POP ↔ 当地运营商 / 用户

# 电商行业 — 电商平台跨区域访问加速

### ** 客户背景**

适用于跨境电商平台及卖家在不同地区访问主站、第三方系统的场景。

### ** 客户需求**

1. 买家访问延迟高、结算慢、购物车卡顿；
2. 卖家后台运维效率低、广告与直播业务受限；
3. 弹性与安全要求高，应对活动流量与攻击风险。

### ** 解决方案**

- **部署 ZGA 全球加速服务**：实现卖家/买家就近访问；
- **SDN 专线加速** 平台与主站间通信；
- **ZDP 安全防护**：四层、七层安全清洗。

### ** 实施架构**

买家/卖家 → ZGA 边缘节点 → ZGA Backbone → 电商主站
（ERP / 支付 / 物流 / 广告 / 第三方平台）

# OTT行业 — 海外OTT盒子

### **客户背景**

面向 OTT 平台（如 Netflix、Disney+、地区性 IPTV），需保证高可用和内容分发安全。

### ** 客户需求**

1. **业务高可靠性**：服务器故障会导致大面积业务中断；
2. **防攻击需求强**：行业内竞争激烈，频发 DDoS 攻击；
3. **成本压力大**：设备与带宽支出高。

### ** 解决方案**

- **在 弹性计算上部署客户源站**：提升容错与弹性；
- **通过 CDN 或自建节点分发**：一跳直连 ISP，降低延迟；
- **ZDP 安全防护**：全局清洗能力。

### **实施架构**

内容源（GPU/VM） → CDN / 自建 POP → 本地用户
（北美源站 → 南美国家节点 → 用户端播放）
