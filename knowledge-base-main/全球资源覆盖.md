# 总结

Zenlayer 已经在全球 **300+ 节点** 部署数据中心和边缘计算节点，覆盖 **六大洲主要城市**。

- 提供 **云网络、裸金属云** 等核心服务
- 与 **主流公有云（AWS、Azure、阿里云、腾讯云、华为云等）** 深度互联
- 支持 **Equinix、Megaport、Coresite** 等交换平台接入
- 能加速 **SaaS 应用（Salesforce、Office365、Github、Zoom）**

通过这些资源，Zenlayer 实现了 **全球范围的低延迟互联和云网融合服务**。每个节点的产品和资源类型有所不同，需要具体查看对应的产品和服务支持区域。
