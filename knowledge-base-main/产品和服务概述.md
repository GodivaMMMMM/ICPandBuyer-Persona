# 计算

- **弹性算力**
  
    定制化虚拟服务器，全球托管您的应用程序或在线业务
    
    简称：ZEC
    
    - **高性能**
        - 采用新一代 Intel / AMD CPU
        - 硬件卸载技术确保网络操作迅速高效
        - 通过 NVMe SSD 和 Converged Ethernet 上的 RDMA (RoCEv2) 实现块存储
    - **连接性**
        - 跨区弹性 IP 分配确保连接不间断
        - 全面支持 IPv4 和 IPv6，实现网络向后兼容
        - 全球所有区域统一接入单一虚拟私有云 (VPC)，实现管理简化
    - **用户体验**
        - 30 秒内即可创建虚拟机
        - 可选择多种 Linux & BSD 以满足灵活性和兼容性需求
- **裸机云**
  
    靠近用户的边缘计算，按需部署
    
    简称：BMC
    
    - **专属服务器**
        - 免受“噪声租户”的干扰
        - 可全面访问硬件特性（如：SGX、硬盘加密）
        - 对于基于 SR-IOV / DPDK 的应用，不影响性能
    - **互联互通**
        - 以 Zenlayer 骨干网连接全球
        - 130+ 边缘节点资源
        - 灵活的 IP 传输能力（流量/带宽）
        - 支持虚拟私有云、私有 VLAN
        - 可与 AWS、Azure、腾讯云等主流公有云互连
    - **自助开通**
        - 10 分钟内开通服务
        - 按需增删带宽和 IP
    - 应用场景
        - 游戏：专用算力，快速部署
        - 流媒体：高速读写，支持大容量存储
        - 电商：高性能计算，全球专属网络
        - 企业应用：性能评估和部署，高性能计算与密集型数据处理
        - CPU/GPU 密集型应用：电影渲染，媒体转码
        - 高安全性应用：硬件隔离，全球专属网络
- **边缘主机托管服务**
  
    提供定制化的机房托管和托管主机解决方案
    
    简称：Colocation
    
    - **解决方案设计**
        - 根据个性化需求进行站点选择
        - 提供详细配置和最佳匹配架构建议
        - 概念验证 (POC) 测试
    - **采购与部署**
        - 设备采购
        - 运维方案交付
        - 协助部署
        - 操作系统和系统/网络配置
        - 网络资源配置
    - **监测与维护**
        - 全天候主动设备和网络监控
        - 利用 Kentik 和 Catchpoint 定制监测工具，提供实时网络分析

# 网络

- **优质互联网带宽**
  
    在边缘和核心区域提供高性能的全球 IP 带宽服务
    
    简称：IPT
    
    - **规模**
        - 全球网络容量超过 130+ Tbps
        - 在全球拥有超过 300 多个联网数据中心，覆盖超过 110 个城市
        - 共拥有超过 2,000+ 台网络设备
    - **性能**
        - 直连各大城市本地 ISP，服务终端用户
        - 全球动态骨干网传输专属骨干实现流通
        - 专业技术团队不断优化网络性能
    - **稳定性**
        - SLA 99.9%
        - 全冗余网络结构
        - 多 ISP 设计，保护上游
- **云网络**
  
    迅速构建全球专属网络至云与数据中心的专属连接，实现超低时延，弹性增减
    
    简称：二层网络（L2，Private Connect，专线）, 三层网络（L3，Cloud Router）
    
    - **高性能**
        - 与公网相比，高峰时段可降低延迟和抖动高达 40%
        - 零丢包
        - 高可用
    - **广泛互联**
        - 快速构建混合云或多云环境，满足业务需求
        - 直连公有云和 QoS
        - 办公室、数据中心和云之间的专用网络
    - **弹性扩容**
        - 按需调整
        - 1 Mbps–10 Gbps
    - 应用场景
        - 游戏：身临其境的游戏体验
        - 视频：畅快明屏，不卡顿
        - 在线教育：音画同步，沉浸式教学
        - 内容提供商：动态加速
        - 多云/混合云：数据同步、快速交互
        - 移动应用：流畅的应用体验

# 应用

- **内容分发网络**
  
    通过分布式交付，灵活处理需求峰值
    
    简称：CDN
    
    - **高性能**
        - 主要城市延迟 < 10 ms
        - 全球其他地区延迟 < 30 ms
        - 主要网络接入点 (PoP) 之间的私有路径，确保跨境流量零丢包
        - NVMe 服务节点，实现极致 I/O
        - 高可用
    - **连接性**
        - 全球私有网络骨干，连接用户
        - 高级路由算法，确保流量优化
        - 广泛的对等 ISP 对等互联，交换高效
    - **灵活性**
        - 按需增/删，适应流量波动需求
        - 灵活计费，实现成本效益最大化
    
- **全球智能加速平台**
  
    建立在 Zenlayer 全球节点和骨干设施上的加速服务。通过专线和骨干智能调度，让应用从源端到终端延迟显著缩短，避免网络波动。提供自动化部署，一键加速全球资源。
    
    简称：ZGA
    
    - **高性能**
        - 主要城市延迟 < 10 ms
        - 全球其他地区延迟 < 30 ms
        - 智能路由，高冗余
        - 多运营商传输链路
    - **定制化**
        - 支持 4–7 层网络协议
        - 协议优化
        - 自主研发 Linux 内核
        - 双向加速
    - **广泛性**
        - 全球网络覆盖
        - 直连各大本地运营商
        - 中国和海外 Zenlayer 全球专用骨干
    - **安全性**
        - 深度网络隔离和负载均衡
        - SSL 加密传输
        - 细粒度策略和白名单选项
    - **应用场景**
        - 游戏：全球服务器加速、推送更新、多人与互动
        - 电商：身份验证、跨平台管理、文件上传
        - 在线教育与直播：互动视频、实时聊天、混合流
        - 公有云加速：跨区域访问、云端管理
        - 物联网：命令传输、数据收集与处理、边缘计算
        - 跨区域办公：SaaS 访问、研发发布、远程办公
        - 远程合作：SSH 访问、文件传输
        - 大数据与灾备：大文件传输、跨区域数据复制
